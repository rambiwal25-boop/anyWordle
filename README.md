# 🎯 AnyWordle

AnyWordle is a modern twist on the classic Wordle game. Instead of using a fixed dictionary, this version allows players to guess *any valid English word*, with real-time validation and definitions powered by the Wiktionary API.

---

## 🚀 Purpose

The goal of AnyWordle is to create a more flexible and educational Wordle experience. Players not only guess words but also learn their meanings, making the game both fun and informative.

---

## 🌐 API Used

This project uses the **Wiktionary API**:

🔗 https://en.wiktionary.org/w/api.php

### What it provides:
- Word validation (check if a word exists)
- Definitions and meanings
- Linguistic data

---

## ✨ Features

- 🔤 Word guessing system (like Wordle)
- ✅ Real-time word validation using API
- 🎨 Color feedback system:
  - Green → correct letter & position  
  - Yellow → correct letter, wrong position  
  - Gray → incorrect letter  
- 📖 Show word meaning after game ends
- 🧠 “Any word” support (not limited to fixed list)
- 💾 Local storage for saving progress
- 📱 Fully responsive UI
